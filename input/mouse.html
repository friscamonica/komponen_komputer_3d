Mouse

Mouse adalah perangkat input yang berfungsi untuk menggerakkan dan mengatur posisi kursor di layar komputer. Pengoperasiannya meliputi beberapa cara, yaitu: menunjuk (point) objek di layar, klik kiri untuk memilih, klik ganda untuk membuka file atau folder, klik kanan untuk menampilkan menu pilihan, serta seret (drag) untuk memindahkan objek dengan menahan tombol kiri sambil menggeser mouse.
