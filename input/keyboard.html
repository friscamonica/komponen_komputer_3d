Keyboard

Keyboard atau papan tombol adalah perangkat input yang digunakan untuk memasukkan huruf, angka, dan simbol ke dalam komputer. Keyboard terdiri dari beberapa kelompok tombol, yaitu: typewriter keys (huruf, angka, tanda baca), function keys (F1â€“F12 untuk fungsi khusus), numeric keypad (angka dan operator aritmatika), cursor control keys (pengatur arah kursor), system keys (tombol sistem), application keys (tombol aplikasi), enter key, dan tombol lainnya seperti delete dan insert.
